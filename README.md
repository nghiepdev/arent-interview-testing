## Getting Started

First, run the development server:

```bash
npm run dev
```

Open `http://localhost:3000` with your browser to see the result.

## Tech stack

- Tailwindcss
- ECharts
- React-hook-form
- Redaxios (same api as Axios)
- React Server Components
- CI/CD for deploy and check code quality (ESLint + Typescript)
